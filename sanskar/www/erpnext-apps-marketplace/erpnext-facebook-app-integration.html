Introduction

This application seamlessly integrates Facebook with ERPNext, allowing users to post, upload, and delete content directly from the ERPNext system.

Key Features

  • Upload and Delete posts from the system.
  • Getting all leads from the Facebook business account.

Setup Meta Developer Account

  • Upload and Delete posts from the system.
  • Generate a new application to obtain the necessary credentials for uploading posts.
  • Click on “Create App” to display the steps for creating an application.
  • Choose the "Other" option to upload posts and get leads from Facebook.
  • After clicking "Next," showcase the options for selecting the type of application.
  • Choose "Business" to access the API for posting images and getting the leads.
  • The next step is to provide the app name and email address. If you have a Facebook business account, select the appropriate account.
  • Setup the marketing API
  • Once you've set up the Marketing API, navigate to the "Graph API Explorer" by clicking on the "Tools" menu in the navbar. From there, you can configure permissions and generate a token.
  • Select all the below-listed permissions as shown in the image below.

Workspace

  • In the Facebook workspace, you can access the Facebook Enquiry, Facebook Post, FaceBook Calender View, FaceBook Enquiry Report, FaceBook Enquiry Report, Facebook Dashboard, Facebook Ads Template, Facebook Dashboard, Facebook Settings, as well as view Cart Views and Line Chart Views.

Facebook Dashboard

  • The dashboard displays a variety of data visualizations, including number cards, line charts, and bar charts, providing a clear and insightful representation of key metrics.

Facebook Settings

  • Enabling the setting grants users access to features for uploading and deleting posts and retrieving leads from Facebook.
  • Obtain the client ID and client secret from the Facebook app settings and set them up here. Fill in all the details here from the generated Graph API explorer.
  • By default, the page access token time limit is set to one hour. To extend this limit, adjust the time limit in the page access token field and click "generate" to extend the access token's duration to 60 days. After generating, this process automatically repeats every 20 days.

Facebook Ads Templates

  • To access Facebook Lead Ads, navigate to the "Facebook Ads Template.
  • To synchronize Lead Ads from Facebook, click on the "Sync Lead Ads" button to retrieve all Lead Ads from Facebook.
  • Click on a specific ID to display the form view. Here, you'll find the Lead Ads Template ID, Name, Status, and Language obtained from Facebook.

Facebook Post

  • To access Facebook posts, navigate to the Facebook Post List. Users can upload and delete posts from the system to Facebook.
  • Click on “Add Facebook Post” to create a new post.
  • Here post has two types:
    • Photo
    • Feed
  • Post Status:
    • Open:- The default post status is set to "open" when a user creates a new post.
    • Published:- If the post is published on Facebook, then the status is automatically set to "Published".
    • Deleted:- If a post is deleted, its status is automatically set to "Deleted".
    • On Hold:- If the system encounters any errors, the status is automatically set to "On Hold".
  • If the post type is "feed," content is uploaded exclusively to Facebook.
  • If the post type is set to "Photo," it provides two options: a message and the ability to attach an image.
  • Users can publish posts on Facebook by clicking the "Publish" button. Once published, the post ID is set in the respective form, and the status is automatically set to “Published”.
  • If a user deletes a post by clicking on the "Delete Post" button, the post is removed from Facebook, and the status is set to "Deleted."

Facebook Enquiry

  • To access "Facebook Enquiry" navigate to the Facebook Enquiry List.
  • Click on the specific ID to display the form view and lead details, which are automatically synchronized from Facebook every 5 minutes using a scheduler.
  • You'll display the basic details filled out by the user on Facebook, such as name, email, mobile number, address, and more.

Facebook Enquiry Report

  • This report provides a comprehensive overview of all Facebook inquiries.
